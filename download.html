Why It's Built This Way

Questions

Before You Install

Is this safe? It's not on the Play Store or App Store.
Yes. MedPREP runs entirely inside your browser's sandbox — the same security model as any website, including your bank's. Installing it just creates a shortcut; it doesn't get deeper system access the way a native app would.
Do I need to register before installing?
You can install the app first, but full question banks unlock after registration (PKR 1,500/session). Register here.
Will it take up storage on my phone?
Barely any — a fraction of what a native app install would use, since content loads from the web and caches only what you actually study.
What if the install button doesn't show up?
Some browsers hide the install prompt. Use Chrome (Android/Windows) or Safari (iOS) for the smoothest experience, or message us on WhatsApp and we'll walk you through it.
Install on iOS
Safari doesn't allow one-tap install — takes 10 seconds manually:
1
Open MedPREP in Safari, then tap the Share icon at the bottom.
2
Scroll down and tap “Add to Home Screen.”
3
Tap Add. The MedPREP icon now opens like a normal app.
