Question 14 of 40 Pharmacology
A 45-year-old patient on warfarin presents with bleeding. Which vitamin antagonizes warfarin's mechanism?
A
Vitamin C
B
Vitamin K ✓
C
Vitamin D
D
Vitamin B12
Explanation
Warfarin inhibits Vitamin K epoxide reductase. Vitamin K reverses this by regenerating active clotting factors.
⚠ Trap: Vitamin C is a common distractor, it has no role in coagulation reversal.
💡 Pearl: Vitamin K works in 6–24 hrs. For immediate reversal → use FFP or PCC.
